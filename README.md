# Unboxing_project: "편의성과 안전성을 강화한 블랙박스"

## 개요
- 운전자의 편의성과 안전성을 강화하기 위해 웹 서버와 모바일 앱의 네트워크 통신을 이용하여 다양한 기능을 제공하는 라즈베리파이 기반 블랙박스 제작 프로젝트

## 기능
- 영상 녹화 및 저장 기능 
- 웹 서비스와 모바일 앱을 통한 영상 확인 및 다운로드 기능
- 차선이탈 방지 기능
- 졸음 방지 기능
- 신고 기능
- 주차위치 알림 기능
