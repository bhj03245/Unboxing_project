# Quick start

# Installation
```Bash
sudo apt-get install wiringpi
```

# Make
```Bash
make camera
```

# Running your RPiCameras with ArduCam
Default: enables and tests all four cameras at compile time (comment out ENABLE_CAMERA_X in arducam_multi_adapter_v2.1_c.cpp to disable)
```Bash
./camera
```
